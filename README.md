Определить структурный тип, описывающий фильмы (название фильма, режиссер,
тип фильма, год выпуска, длительность, стоимость создания, рейтинг на
Кинопоиске, главный актер). Заполнить структурный массив 20-ю записями.
Переписать из исходного массива в другой массив, информацию о фильмах, в
которых снималась Николь Кидман. Затем новый массив отсортировать по названию
по алфавиту. Вывести все данные по конкретному фильму. Вывести 6 самых дорогих
в производстве фильма за 2000-е годы. Реализовать функцию изменения фильма. В
отдельный массив поместить все триллеры, в которых режиссера зовут Джон, но это
разные режиссеры. Реализовать вывод отфильтрованных данных в виде оберточной
функции.

Требования к решению:
1. Использовать функции для каждой подзадачи.

(enum).

структуру.
4. Ввод данных осуществить в виде инициализации массива структур
внутри программы (не с клавиатуры).
5. Алгоритм сортировки – пузырьком.
6. После выполнения следующего занятия выложить решение на github.
